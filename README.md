En mobilapp byggd med React Native som hjälper användare att hålla koll på sina uppgifter och scheman.

## Funktioner
- Söka på valfria städer för att se väder
- Kunna lägga till städer i favoritlista för att snabbt kunna navigera till dem
- Enkel navigering mellan vyerna med tabs

## Tekniker
- React Native
- Expo
- TypeScript
